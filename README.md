# Elementor Plyr Community Tracker

This repository is the dedicated space for community feedback, bug reports, and feature requests for the [Elementor Plyr](https://babakfp.gumroad.com/l/elementor-plyr) plugin.

- Note: This repo does not contain the source code. The core code is hosted privately.
- Bugs: Use the Issues tab to report problems.
- Ideas & Chat: Use the Discussions tab for feature requests and general community conversation.
